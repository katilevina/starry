# Your Stories

This folder stores your personal stories in STARR format.

***

## Getting Started

1. Copy `../template_story.md` into this folder
2. Rename to `story_[number]_[descriptive_slug].md`
3. Fill in all sections
4. Add entry to `stories_index.md` under the correct company

***

## stories_index.md

The index is a **table of contents**, not a search tool. It shows what stories exist, which companies, which periods, and brief context.

Real skill matching happens during `/map-skills` when the full story files are analyzed — the same story can support many different skills depending on the target role.

To create or rebuild the index, use `../template_stories_index.md` as reference.

***

## Important Notes

* Use chronological numbering: story_01, story_02, etc.
* All stories should link to companies you worked for
* Include metrics and measurable results in the story itself
* Update index after adding each story

# Your Achievement Stories

This folder is for storing your personal achievement stories in STARR format.

## Getting Started

1. Copy the template from `../template_achievement.md`
2. Rename to `story_[descriptive_slug].md` (e.g., `story_data_framework.md`)
3. Fill in your achievement details
4. **Update `stories_index.md`** to include your new story

## About stories_index.md

The `stories_index.md` file is a catalog of all your achievement stories.

**What to include:**
- Story title and link
- Company name
- Dates
- Brief summary

**Example format:**
```markdown
## Story Catalog

### 2019
- [Process Standardization & Team Scaling](story_05_process_standardization_team_scaling_doczilla.md) — Doczilla (Apr-Dec 2019)
```

## Important Notes

- ✅ Keep your stories here - they're personal and private
- ✅ Use chronological numbering: story_01, story_02, etc.
- ✅ All stories should link to companies you worked for
- ✅ Include metrics and measurable results
- ✅ Update index after adding each story

**Questions?** See the main project README or run `/add-achievement` command.

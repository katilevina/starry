## Workflow

**When applying with a target role CV:**

1. Say "I applied to [Company]" → Claude snapshots CV from `target_roles/my_data/role_[name]/cv.md` to `app_[company]_[date]/cv_sent.md`
2. Send final text you actually submitted → Claude compares with role version and updates snapshot
3. Add entry here, update status

**When applying without a target role (volunteer, networking):**

1. CV created directly in `app_[company]_[date]/cv_sent.md`
2. Add entry here, update status

**Status updates:**

* When status changes → update Status, Last update, and recalculate funnel above

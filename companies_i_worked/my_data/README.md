# Your Company Profiles

This folder is for storing profiles of companies you've worked for.

## Getting Started

1. Copy the template from `../template_company.md`
2. Rename to `company_[slug].md` (e.g., `company_acme_corp.md`)
3. Fill in company details
4. **Update `companies_index.md`** to include the new company

## About companies_index.md

The `companies_index.md` file is a catalog of all companies you've worked for.

**What to include:**
- Company name and link
- Employment dates
- Role
- Brief description

**Example format:**
```markdown
## Companies Catalog

### Current
- [Volkswagen Financial Services](company_volkswagen_financial_services.md) — 2015-2019

### Previous
- [Doczilla](company_doczilla.md) — 2019
```

## Important Notes

- ✅ Company profiles are built from your achievements (not the other way around)
- ✅ Update company profiles when you add new achievements
- ✅ Include links to related achievement stories
- ✅ Keep the index current

**Questions?** See the main project README or run `/add-company` command.

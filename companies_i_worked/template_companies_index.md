## 📝 How to Add Companies

Use the `/add-company` command to interactively create a new company profile.

**Important:** Company profiles are built FROM your STARR stories, not the other way around.

The command will:

1. Check if you have stories for this company
2. **If yes:** Extract responsibilities, tools, achievements from your stories automatically
3. **If no:** Encourage you to add stories first (or create minimal profile with warning)
4. Ask only for high-level company context (industry, business model, size)
5. Link all related stories automatically
6. Create the file and update this index

***

## 🔗 Cross-Reference

**Stories link to companies** → Each story file has a company header
**Companies link to stories** → Each company file lists related stories

**For CV generation:**

* Company profiles provide organizational context
* Stories provide the detailed evidence
* Always aligned because companies are built FROM stories
